100% data-eigendom

Zo werkt {{ .Site.Title }}

{{ .Site.Title }} heeft een unieke manier om ruimtelijke gegevens vast te leggen: uw gegevens worden rechtstreeks op uw eigen GIS server opgeslagen via het open protocol Web Feature Service (WFS-T).

Gegevens worden hierdoor gedecentraliseerd opgeslagen, veilig en in eigen beheer. Precies op de plek waar u dit wilt. Via open standaarden kunt u andere software integreren in uw eigen datacenter. Bij {{ .Site.Title }} staat datasoevereiniteit voorop.

Analyseer

De vastgelegde gegevens gelden als één enkele bron van waarheid voor gebruik in uw hele organisatie.

Functionaliteit

Eén tool voor alle gegevens

{{ .Site.Title }} biedt één gebruiksvriendelijke tool voor alle typen geo-data. Elke gebruiker kan zonder instructies deze app overal gebruiken. De meeste van onze gebruikers stellen hun formulieren op en trainen hun personeel in minder dan een dag.

De gegevens worden eerst tijdelijk opgeslagen op het apparaat waarmee in het veld wordt ingewonnen. Zo zijn de gegevens - ook bij geen verbinding - altijd gewaarborgd. Wanneer er een connectie is met het internet, worden de gegevens doorgestuurd en bewaard op het datacenter.
